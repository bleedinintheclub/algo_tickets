## 11. Алгоритм Беллмана-Форда

### Идея

Алгоритм позволяет найти кратчайшие расстояния от стартовой вершины до всех остальных в графе. Может работать с отрицательными весами ребер. 
Использует релаксацию, после ее выполнения по всем ребрам массив `dist` содержит кратчайшие расстояния, если в графе нет отрицательных циклов, достижимых из стартовой вершины.

### Релаксация

**Релаксация** - атомарная операция в алгоритмах кратчайших путей. 
Релаксацией ребра `(u, v)` с весом `w` называют операцию `dist[v] = min(dist[v], dist[u] + w)`. 
При уменьшении `dist[v]` обновляют `parent[v] = u` для корректного восстановления пути в дальнейшем.

### Принцип работы

1. Инициализация: `dist[s] = 0`, `dist[v_i] = INF`, `parent[v_i] = -1`
2. Повторить `V-1` раз:
    - Для каждого ребра выполнить релаксацию
    - Более ранний выход из цикла, если в проходе не произошло ни одного обновления
3. Проверка на наличие отрицательного цикла: 
- сделать еще один проход по всем ребрам
- если существует ребро, которое можно релаксировать - найден отрицательный цикл, достижимый из стартовой вершины

### Структруры данных

- Список ребер: `vector<tuple<int, int, long long>>` 
- Массив dist: `vector<long long> dist(n)` - хранит текущие оценки расстояний
- Массив parent: `vector<int> parent(n)` - массив предков для восстановления кратчайшего пути

### Сложность

- Временная: `O(V * E)` - (V-1) итераций по E ребрам
- Пространственная: `O(V + E)`

### Реализация на C++

```cpp
const long long INF = std::numeric_limits<long long>::max();

void bellman_ford(int n
                    , const std::vector<std::tuple<int, int, long long>>& adj
                    , int s
                    , std::vector<long long>& dist
                    , std::vector<int>& parent) {

    dist.assign(n, INF);
    parent.assign(n, -1);
    dist[s] = 0;

    // V-1 итераций релаксации
    for (int iter = 0; iter < n; ++iter) {
        bool updated = false;  // для возможности раннего выхода
        for (const auto& e : adj) {
            int u, v;
            long long w;
            std::tie(u, v, w) = e;
            if (dist[u] != INF && dist[v] > dist[u] + w) {
                dist[v] = dist[u] + w;
                parent[v] = u;
                updated = true;
            }
        }
        if (!updated) break;
    }
}
```
