## Жадный алгоритм для поиска раскраски графа

### Идея

Алгоритм раскрашивает граф в минимальное количество цветов так, что никакие две смежные вершины не имеют одного цвета. Проходим граф и присваиваем каждой вершине первый допступный цвет. 

### Принцип работы
1. Обходим вершины в порядке убывания степеней. 
2. Первая вершина красится в цвет `0`, остальные - `-1`.
3. На каждой итерации (для каждой вершины):
    - проверяем соседей
    - составляем список занятых цветов
    - присваиваем вершине минимальный из доступных цветов

### Сложность

Временная:
1. определяем степени: `O(V)`
2. сортируем вершины: `O(VlogV)`
3. жадный алгоритм раскраски: `O(V + E)`
Итого: `(E + VlogV)`  

Пространственная: `O(V)` 


### Псевдокод

```python
degrees = calculate_degrees(graph)

colors = [-1] * V  

for u in sorted_vertices:
    used_colors = set()
    for neighbor in adj[u]:
        if colors[neighbor] != -1:
            used_colors.add(colors[neighbor])
    
    color = 0
    while color in used_colors:
        color += 1
    
    colors[u] = color
```
