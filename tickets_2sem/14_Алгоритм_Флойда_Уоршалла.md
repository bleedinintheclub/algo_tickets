## 14. Алгоритм Флойда-Уоршалла

### Идея

Алгоритм находит кратчайшие пути от всех вершин до всех. Может работать при отрицательном весе ребер. Работает через динамическое программирование.

### Принцип работы

1. Инициализируем матрицу расстояний `dist[i][j] = INF` и матрицу `next[i][j]` для восстановления пути `next[i][j]` 
(`next[i][j] = j`, если есть дуга из `i` в `j`). 
2. Для `k = 0..n-1`: для всех пар `(i, j)`: если `dist[i][k] + dist[k][j] < dist[i][j]`, 
то `dist[i][j] = dist[i][k] + dist[k][j]`.
3. После завершения: `dist[i][j]` - длина кратчайшего пути из `i` в  `j`. 

При релаксации также обновляем матрицу `next`: `next[i][j] = next[i][k]`.

### Структуры данных

- Матрица расстояний
- Матрица предков
- Список смежности

### Сложность

- Временная: `O(V^3)` - три вложенных цикла 
- Пространственная: `O(V^2)`

### Уточнение

Если после работы алгоритма `dist[i][j] < 0`, значит в графе есть отрицательный цикл, достижимый из вершины `i`. 

### Восстановление пути

- Если `next[i][j] = -1`, то пути нет
- Иначе путь из `i` в `j` восстанавливается:
```cpp
std::vector<int> path;
int curr = i;
while (curr != j) {
    curr = next[curr][j];
    if (curr == -1) break; // отрицательный цикл
    path.push_back(curr);
}
```

### Реализация на C++

```cpp
const long long INF = std::numeric_limits<long long>::max();

void floyd_warshall(int n
                    , std::vector<std::vector<int>>& dist
                    , std::vector<std::vector<int>>& next) {

    for (int k = 0; k < n; ++k) {
        for (int i = 0; i < n; ++i) {
            if (dist[i][k] == INF) continue;
            for (int j = 0; j < n; ++j) {
                if (dist[k][j] == INF) continue;
                
                if (dist[i][k] + dist[k][j] < dist[i][j]) {
                    dist[i][j] = dist[i][k] + dist[k][j];
                    next[i][j] = next[i][k];
                }
            }
        }
    }

}
```
