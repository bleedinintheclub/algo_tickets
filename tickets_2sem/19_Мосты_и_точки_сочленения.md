## Поиск мостов и точек сочленения

### Теория

**Мост** — ребро, при удалении которого граф становится несвязным. 
**Точка сочленения** — вершина, при удалении которой увеличивается количество 
компонент связности.

### Нахождение мостов

1. Запускается DFS. Для каждой вершины вычисляется:
    - `tin[v]` - время входа (порядковый номер при первом посещении)
    - `low[v]` - минимальное `tin`, до которого можно добраться из вершины `v`.
2. Если при для ребра `(v1, v2)` верно, что `tin[v1] < low[v2]`, то это ребро - мост.
3. Для обработки кратных ребер следует ввести индентификатор ребра.

### Нахождение точек сочленения

1. Запускается DFS. Для каждой вершины вычисляется:
    - `tin[v]` - время входа (порядковый номер при первом посещении)
    - `low[v]` - минимальное `tin`, до которого можно добраться из вершины `v`.
2. 
```cpp
#include <vector>
using namespace std;

vector<vector<int>> adj;
vector<int> tin, low;
vector<bool> visited;
int timer;

void dfs(int v, int p = -1) {
    visited[v] = true;
    tin[v] = low[v] = timer++;

    int children = 0;

    for (int to : adj[v]) {
        if (to == p) continue;

        if (visited[to]) {
            // back edge
            low[v] = min(low[v], tin[to]);
        } else {
            dfs(to, v);

            low[v] = min(low[v], low[to]);

            // МОСТ
            if (low[to] > tin[v]) {
                // (v, to) — мост
            }

            // ТОЧКА СОЧЛЕНЕНИЯ
            if (low[to] >= tin[v] && p != -1) {
                // v — точка сочленения
            }

            children++;
        }
    }

    // отдельный случай для корня
    if (p == -1 && children > 1) {
        // v — точка сочленения
    }
}
```
