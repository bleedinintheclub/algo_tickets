## 13. Алгоритм Дейкстры 

### Идея

Алгоритм позволяет найти кратчайшие расстояния от стартовой вершины до всех остальных в графе, вес ребер которого неотрицательный.
Каждый из путей можно сохранить через массив предков.

### Релаксация

**Релаксация** - атомарная операция в алгоритмах кратчайших путей. 
Релаксацией ребра `(u, v)` с весом `w` называют операцию `dist[v] = min(dist[v], dist[u] + w)`. 
При уменьшении `dist[v]` обновляют `parent[v] = u` для корректного восстановления пути в дальнейшем.

### Принцип работы

1. Инициализируем `dist[start] = 0`, для остальных вершин `dist[v_i] = inf`, `parent[v_i] = -1`.
2. Пока есть неоработанные вершины:
    1. Выбираем необработанную вершину `u` с минимальным текущим `dist[u]`.
    2. Помечаем ее как обработанную (считаем `dist[u]` финальным, дальше не будем его менять).
    3. Для каждого ребра выполняем релаксацию 

### Инвариант

Когда вершина `u` помечена как обработанная, текущее `dist[u]` считается равно истинному кратчайшему расстоянию от `s` до `u`. 
Доказательство от противного: если бы `dist[u]` было бы не минимальным, то тогда существовал бы путь от `s` до `u` через необработанную вершину `x`. Но тогда `dist[x] < dist[u]`, и мы бы обработали `x` раньше `u`.

### Ограничения

Вес всех ребер должен быть `>= 0`. При отрицательных весах алгоритм не гарантирует корректный результат. (см Беллмана-Форда и др.)

### Структуры данных и варианты реализации

1. Приоритная очередь
- `adj list`: `vector<vector<pair<v, w>>>`
- _min heap_ хранит пары (dist, vertex)
- decrease-key (ленивая реализация): при улучшении `dist[v]` не ищется существующая запись, чтобы сделать `sift_up`, просто вставляем новую пару `(new_dist, v)` и просеиваем уже ее наверх, а устаревшая запись остается в куче. При чтении кучи по массиву `dist` определяется, устаревшая ли запись. Но при таком подходе растет использование памяти за каждую релаксацию. 

2. Массив (линейный поиск минимума)
- массив `dist`, массив `used`: в каждой из `V` итераций выбираем необработанную вершину с минимальным `dist` за `O(V)`, релаксируем ее ребра. 

### Сложность 

1. Priority queue
- Временная: `O(V + E)logV` = `O(VlogV)`(V * поиск min_dist) + `ElogV`(E * релаксация)
- Пространственная: `O(V + E)`

2. Array
- Временная: `O(V^2)`
- Пространственная: `O(V + E)`

### Реализация на C++

#### priority_queue, lazy decrease_key

```cpp
const long long INF = std::numeric_limits<long long>::max();

void dijkstra_lazy(int n
                    , const std::vector<std::vector<std::pair<int, long long>>>& adj
                    , int s      // стартовая вершина
                    , std::vector<long long>& dist
                    , std::vector<int>& parent) {
    
    dist.assign(n, INF);
    parent.assign(n, -1);
    std::vector<bool> used(n, false);

    // min-heap
    std::priority_queue<
        std::pair<long long, int>,
        std::vector<std::pair<long long, int>>,
        std::greater<std::pair<long long, int>>
    > pq;

    dist[s] = 0;
    pq.push({0, s});

    while (!pq.empty()) {
        auto top = pq.top();
        pq.pop();
        long long d = top.first;
        int v = top.second;

        // пропускаем устаревшую запись
        if (d != dist[v]) continue;
        if (used[v]) continue;
        

        for (const auto& e : adj[v]) {
            int neighbor = e.first;
            long long w = e.second;
            if (dist[neighbor] > dist[v] + w) {
                dist[neighbor] = dist[v] + w;  // релаксация
                parent[neighbor] = v;
                pq.push({dist[neighbor], neighbor});  // decrease key
            }
        }
    }
}
```

#### array

```cpp
const long long INF = std::numeric_limits<long long>::max();

void dijkstra_arr(int n
                    , const std::vector<std::vector<std::pair<int, long long>>>& adj
                    , int s
                    , std::vector<int> dist
                    , std::vector<int>& parent) {

    dist.assign(n, INF);
    parent.assign(n, -1);
    std::vector<bool> used(n, false);

    dist[s] = 0;

    for (iter = 0; iter < n; ++iter) {
        int v = -1;
        long long best = INF;
        // поиск необработанной вершины с мин dist
        for (int i = 0; i < n; ++i) {
            if (!used[v] && dist[i] < best) {
                best = dist[i];
                v = i;
            }
        }
        if (v == -1) break;

        used[v] = true;

        // релаксация ребер из v
        for (const auto& e : adj[v]) {
            int neighbor = e.first; 
            long long w = e.second; 
            if (dist[neighbor] < dist[v] + w) {
                dist[neighbor] = dist[v] + w;
                parent[neighbor] = v;
            }
        }
    }
}
```
