## 5. Поиск и восстановление всех видов циклов

### Теория 
**Замкнутый путь** - путь, у которого начало и конец совпадают.
**Цепь** - путь без повторяющихся ребер.
**Простая цепь** - путь без повторяющихся вершин.
**Цикл** - замкнутая цепь.
**Эйлеров путь** - путь, проходящий через все ребра графа по 1 разу.
**Гамильтонов путь** - путь, проходящий через все вершины графа по 1 разу.

### Виды циклов:
- **Простой цикл** - замкнутая простая цепь, то есть замкнутый путь без повторяющийся вершин (n>=3). 
Алгоритм: используем DFS с тремя цветами, из найденных циклов выделяем простые.

- **Эйлеров цикл** - замкнутый Эйлеров путь.
Алгоритмы: Флери, Хильхорцера

- **Гамильтонов цикл** - замкнутый Гамильтонов путь.
Алгоритм: полный перебор (NP-полная задача).

### Принципы работы
- **DFS с тремя цветами**
1. Инициализация: все вершины окрашиваются в белый цвет.
2. Выбираем любую белую вершину и запускаем с нее рекурсивный алгоритм.
3. При входе в белую вершину она окрашивается в серый. Это означает, что вершина сейчас находится в процессе обработки.
4. Рекурсия по соседям:
    1. Сосед белый: рекурсивный вызов.
    2. Сосед серый: обнаружено обратное ребро, в графе есть цикл, записываем.
    3. Сосед черный: continue (уже обработано).
5. Когда все исходящие пути из текущей вершины полностью исследованы, она становится черной. 

- **Алгоритм Флери**
1. Проверка критериев эйлеровости.
2. Начинаем с любой вершины (в случае поиска пути - с вершины с нечетной степенью).
3. Выбираем ребро по правилу:  не проходить по мосту, если есть другие доступные ребра.
4. Переходим по выбранному ребру и "удаляем" его из графа.
5. Новая вершина становится текущей. 
    
- **Алгоритм Хирхольцера**
1. Проверка критериев эйлеровости.
2. Начинаем с любой вершины.
3. Поиск первого цикла: идем из вершины `v` по ребрам произвольно, удаляя пройденные, пока не вернемся в `v`.
4. Расширение: Выбираем в цикле любую вершину, у которой есть инцидентные (еще не удаленные ребра). Из этой вершины запускаем поиск цикла из `3)`.
5. Склеивание: вставляем новый найденный цикл в старый на месте вершины.
6. Повторяем шаги `4)`, `5)`, пока не закончатся ребра.

- **Поиск гамильтонова цикла**
1. Выбираем произвольную вершину.
2. Пробуем добавить в путь соседнюю вершину, если она не была посещена:
3. Проверка:
    - если путь содержит все `V` вершин и последняя вершина совпадает с начальной - цикл найден
    - если мы зашли в тупик (нет непосещенных соседей, а вершины пройдены не все), то делаем `back track` (шаг назад) к предыдущей вершине и пробуем другого соседа.
4. Если перебраны все варианты и цикл не найден - значит его нет. 

### Сложность работы
- **DFS с тремя цветами**
Временная: `O(V + E)`
Пространственная: `V`

- **Флери**
Временная: `O(E^2)`
Пространственная: `O(V + E)` для проверки ребра на мост и хранение пути
#### Анализ сложности
**Внутри одной проверки:**
1. DFS → $O(V + E)$
2. Удаление ребра
3. DFS → $O(V + E)$
4. Возвращение ребра
👉 Итого одна проверка: **$O(V + E)$**
**Сколько раз это делается:**
Мы проходим по всем рёбрам, что дает примерно **$E$** шагов. На каждом шаге можем проверять несколько рёбер, но в среднем считаем как $E$.
👉 Итог: **$E \cdot (V + E)$**
**Упрощение:**
В графах обычно $E \ge V$, значит: **$O(E \cdot (V + E)) \approx O(E^2)$**


- **Хиркольцера**
Временная: `O(E)` `O(V + E)`
Пространственная: `O(E)` - результат, `O(V)` - стэк, res + st = `O(V + E)`

- **Поиск гамильтонова цикла**
Временная: `O(V!)`
Пространственная: `O(V)`

- **Поиск гамильтонова цикла**
```python
path = [-1] * V
path[0] = 0 # Начинаем с 0-й вершины

def solveHamiltonian(pos):
    # Базовый случай: все вершины в пути
    if pos == V:
        # Проверяем, есть ли ребро из последней вершины в первую
        return has_edge(path[pos-1], path[0])
    
    for v in range(1, V):
        if is_safe(v, pos):
            path[pos] = v
            
            if solveHamiltonian(pos + 1):
                return True
                
            # Если не подошло - откат (Backtrack)
            path[pos] = -1
            
    return False

def is_safe(v, pos):
    # 1. Есть ли ребро между предыдущей и текущей
    if not has_edge(path[pos-1], v):
        return False
    # 2. Не была ли эта вершина уже использована в пути
    if v in path:
        return False
    return True
```

### Реализация на C++

#### DFS с тремя цветами
Алгоритм ищет все циклы в неориентированном графе (необязательно связном). Можно модифицировать алгоритм для работы в орграфе, убрав проверку на `parent`.

```cpp
class Graph {
    int V;
    std::vector<std::vector> adj;
    enum Color {
        White,
        Gray,
        Black
    };

    void find_cycles(int v,         // текущая вершина
                    int parent,         // родительская вершина (откуда пришли)  
                    std::vector<Color>& color,      // массив цветов вершин
                    std::vector<int>& path,         // текущий путь обхода (серые вершины)
                    std::vector<std::vector<int>>& all_cycles)          // найденные циклы
    {
        color[v] = Gray;        // вершина в процессе обработки
        path.push_back(v);    // path хранит последовательность вершин от начала обхода до текущей

        // просматриваем всех соседей текущей вершины
        for (int neighbor : adj[v]) {
            if (neighbor == parent) {
                continue;           // две симм. дуги за цикл не считаем
            }

            if (color[neighbor] == White) {  // сосед не был посещен
                find_cycles(int neighbor, v, color, path, all_cycles); // продолжаем обход
            }

            else if (color[neighbor] == Gray) {  // найден цикл (вершина уже была посещена)
                std::vector<int> cycle;

                // находим в path позицию, с которой цикл начался
                auto it = std::find(path.begin(), path.end(), neighbor);
                for (; it < path.end(); it++) {
                    cycle.push_back(*it);
                }
                all_cycles.push_back(cycle);  // сохраняем цикл
            }
        }
        color[v] = Black; // завершаем обработку вершины
        path.pop_back();
    }

public:
    std::vector<std::vector<int>> list_cycles() {
        std::vector<std::vector<int>> all_cycles;
        std::vector<Color> color(V, White);
        std::vector<int> path;

        for (int i = 0; i < V; i++) {
            if (color[i] == White) {
                // parent = -1 означает, что у стартовой вершины нет родителя
                find_cycles(i, -1, color, path, all_cycles);
            }
        }
        return all_cycles;
    }

};
```
#### Алгоритм Флёри
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

class FleuryGraph {
private:
    int V;
    std::vector<std::vector<int>> adj;

    void removeEdge(int u, int v) {
        auto it = std::find(adj[u].begin(), adj[u].end(), v);
        if (it != adj[u].end()) adj[u].erase(it);

        it = std::find(adj[v].begin(), adj[v].end(), u);
        if (it != adj[v].end()) adj[v].erase(it);
    }

    void addEdgeBack(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    int dfsCount(int v, std::vector<bool>& visited) const {
        visited[v] = true;
        int count = 1;
        for (int to : adj[v]) {
            if (!visited[to]) {
                count += dfsCount(to, visited);
            }
        }
        return count;
    }

    bool isValidNextEdge(int u, int v) {
        if (adj[u].size() == 1) {
            return true;
        }

        std::vector<bool> visitedBefore(V, false);
        int countBefore = dfsCount(u, visitedBefore);

        removeEdge(u, v);

        std::vector<bool> visitedAfter(V, false);
        int countAfter = dfsCount(u, visitedAfter);

        addEdgeBack(u, v);

        return countBefore == countAfter;
    }

    void fleuryDfs(int u, std::vector<std::pair<int, int>>& path) {
        std::vector<int> neighbors = adj[u];
        for (int v : neighbors) {
            if (std::find(adj[u].begin(), adj[u].end(), v) != adj[u].end() &&
                isValidNextEdge(u, v)) {
                path.push_back({u, v});
                removeEdge(u, v);
                fleuryDfs(v, path);
            }
        }
    }

    bool isConnectedIgnoringIsolated() const {
        std::vector<bool> visited(V, false);
        int start = -1;

        for (int i = 0; i < V; ++i) {
            if (!adj[i].empty()) {
                start = i;
                break;
            }
        }

        if (start == -1) return true;

        std::vector<int> stack = {start};
        visited[start] = true;

        while (!stack.empty()) {
            int u = stack.back();
            stack.pop_back();
            for (int v : adj[u]) {
                if (!visited[v]) {
                    visited[v] = true;
                    stack.push_back(v);
                }
            }
        }

        for (int i = 0; i < V; ++i) {
            if (!adj[i].empty() && !visited[i]) {
                return false;
            }
        }

        return true;
    }

public:
    explicit FleuryGraph(int vertices) : V(vertices), adj(vertices) {}

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    std::vector<std::pair<int, int>> findEulerPathOrCycle() {
        if (!isConnectedIgnoringIsolated()) {
            return {};
        }

        int oddCount = 0;
        int start = 0;
        for (int i = 0; i < V; ++i) {
            if (adj[i].size() % 2 != 0) {
                ++oddCount;
                start = i;
            }
        }

        if (oddCount != 0 && oddCount != 2) {
            return {};
        }

        if (oddCount == 0) {
            for (int i = 0; i < V; ++i) {
                if (!adj[i].empty()) {
                    start = i;
                    break;
                }
            }
        }

        std::vector<std::pair<int, int>> path;
        fleuryDfs(start, path);
        return path;
    }
};

int main() {
    FleuryGraph g(4);
    g.addEdge(0, 1);
    g.addEdge(1, 2);
    g.addEdge(2, 0);
    g.addEdge(0, 3);
    g.addEdge(3, 2);

    auto path = g.findEulerPathOrCycle();

    if (path.empty()) {
        std::cout << "Euler path/cycle does not exist\n";
    } else {
        for (auto [u, v] : path) {
            std::cout << u << " -> " << v << '\n';
        }
    }
}
```

```cpp
#include <iostream>
#include <vector>
#include <stack>
#include <algorithm>

class HierholzerGraph {
private:
    int V;
    std::vector<std::vector<int>> adj;

    void removeEdge(int u, int v) {
        auto it = std::find(adj[u].begin(), adj[u].end(), v);
        if (it != adj[u].end()) adj[u].erase(it);

        it = std::find(adj[v].begin(), adj[v].end(), u);
        if (it != adj[v].end()) adj[v].erase(it);
    }

    bool isConnectedIgnoringIsolated() const {
        std::vector<bool> visited(V, false);
        int start = -1;

        for (int i = 0; i < V; ++i) {
            if (!adj[i].empty()) {
                start = i;
                break;
            }
        }

        if (start == -1) return true;

        std::stack<int> st;
        st.push(start);
        visited[start] = true;

        while (!st.empty()) {
            int u = st.top();
            st.pop();

            for (int v : adj[u]) {
                if (!visited[v]) {
                    visited[v] = true;
                    st.push(v);
                }
            }
        }

        for (int i = 0; i < V; ++i) {
            if (!adj[i].empty() && !visited[i]) {
                return false;
            }
        }

        return true;
    }

public:
    explicit HierholzerGraph(int vertices) : V(vertices), adj(vertices) {}

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u);
    }

    std::vector<int> findEulerPathOrCycle() {
        if (!isConnectedIgnoringIsolated()) {
            return {};
        }

        int oddCount = 0;
        int start = 0;
        for (int i = 0; i < V; ++i) {
            if (adj[i].size() % 2 != 0) {
                ++oddCount;
                start = i;
            }
        }

        if (oddCount != 0 && oddCount != 2) {
            return {};
        }

        if (oddCount == 0) {
            for (int i = 0; i < V; ++i) {
                if (!adj[i].empty()) {
                    start = i;
                    break;
                }
            }
        }

        std::vector<std::vector<int>> g = adj;
        std::stack<int> st;
        std::vector<int> result;

        st.push(start);

        while (!st.empty()) {
            int u = st.top();
            if (!g[u].empty()) {
                int v = g[u].back();
                g[u].pop_back();

                auto it = std::find(g[v].begin(), g[v].end(), u);
                if (it != g[v].end()) g[v].erase(it);

                st.push(v);
            } else {
                result.push_back(u);
                st.pop();
            }
        }

        std::reverse(result.begin(), result.end());
        return result;
    }
};

int main() {
    HierholzerGraph g(4);
    g.addEdge(0, 1);
    g.addEdge(1, 2);
    g.addEdge(2, 0);
    g.addEdge(0, 3);
    g.addEdge(3, 2);

    auto cycle = g.findEulerPathOrCycle();

    if (cycle.empty()) {
        std::cout << "Euler path/cycle does not exist\n";
    } else {
        for (int v : cycle) {
            std::cout << v << ' ';
        }
        std::cout << '\n';
    }
}
```

#### Поиск гамильтонова цикла
```cpp
#include <iostream>
#include <vector>

class HamiltonianGraph {
private:
    int V;
    std::vector<std::vector<int>> adjMatrix;

    bool isSafe(int v, const std::vector<int>& path, const std::vector<bool>& used, int pos) const {
        if (adjMatrix[path[pos - 1]][v] == 0) {
            return false;
        }
        if (used[v]) {
            return false;
        }
        return true;
    }

    bool backtrack(std::vector<int>& path, std::vector<bool>& used, int pos) const {
        if (pos == V) {
            return adjMatrix[path[pos - 1]][path[0]] == 1;
        }

        for (int v = 1; v < V; ++v) {
            if (isSafe(v, path, used, pos)) {
                path[pos] = v;
                used[v] = true;

                if (backtrack(path, used, pos + 1)) {
                    return true;
                }

                used[v] = false;
                path[pos] = -1;
            }
        }

        return false;
    }

public:
    explicit HamiltonianGraph(int vertices)
        : V(vertices), adjMatrix(vertices, std::vector<int>(vertices, 0)) {}

    void addEdge(int u, int v) {
        adjMatrix[u][v] = 1;
        adjMatrix[v][u] = 1;
    }

    std::vector<int> findHamiltonianCycle() const {
        std::vector<int> path(V, -1);
        std::vector<bool> used(V, false);

        path[0] = 0;
        used[0] = true;

        if (!backtrack(path, used, 1)) {
            return {};
        }

        path.push_back(0);
        return path;
    }
};

int main() {
    HamiltonianGraph g(5);
    g.addEdge(0, 1);
    g.addEdge(1, 2);
    g.addEdge(2, 3);
    g.addEdge(3, 4);
    g.addEdge(4, 0);
    g.addEdge(1, 3);

    auto cycle = g.findHamiltonianCycle();

    if (cycle.empty()) {
        std::cout << "Hamiltonian cycle does not exist\n";
    } else {
        for (int v : cycle) {
            std::cout << v << ' ';
        }
        std::cout << '\n';
    }
}
```
