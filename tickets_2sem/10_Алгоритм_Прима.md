## 10. Алгоритм Прима

### Идея

Жадный алгоритм построения минимального остовного дерева (MST) в связном неориентированном взвешенном графе.
Алгоритм начинает с некоторой стартовой вершины и растит дерево путем последовательного добавления ребра с минимальным весом, соединяющего множество вершин в MST с остальными. В отличие от алгоритма Краскала, Прима поддерживает одну растущую компоненту и расширяет ее по одному ребру.

## Теория

**Минимальное остовное дерево (MST)** - это ациклический связный подграф, в который 
входят все его вершины, обладающий минимальным суммарным весом ребер.

**лемма о безопасном ребре**

## Принцип работы

через priority queue 
1. Выбрать стартовую вершину, пометить как включенную в MST
2. Положить в приоритетную очередь все ребра, которые выходят из MST в невключенные в него вершины. Элемент очереди - (вес, вершина, родитель).
3. Достать из очереди ребро с минимальным весом: 
    - Если вершина уже включена - пропустить.
    - Иначе: включить вершину в MST, добавить ребро в результат и положить в очередь все ребра из этой вершины в еще невключенные вершины.
4. Повторять, пока не будут включены все вершины или пока очередь не станет пустой.
5. Если в результате включено меньше `n - 1` ребер - граф несвязен.

### Сложность

1. Временная: 
    - `O(E)` вставок ребра в кучу
    - `O(logE)` - операцияизвлечения из кучи
    Итого: `O(ElogV)`
2. Пространственная: `O(V + E)`

### Реализация на C++

```cpp
struct Edge {
    int to;  
    long long w;
};

struct Item {
    long long w;
    int v;
    int p; 
    bool operator<(const Item& other) const {
        return w < other.w;
    }
};

int main() {
    int n, m;

    std::vector<std::vector<Edge>> g(n + 1);
    for (int i = 0; i < m; ++i) {
        int u, v;
        ll w;
        cin >> u >> v >> w;
        g[u].push_back({v, w});
        g[v].push_back({u, w});
    }

    std::vector<char> used(n + 1, false);
    std::vector<int> parent(n + 1, 0);
    std::vector<long long> edge_w(n + 1, 0);

    std::priority_queue<Item> pq;
    pq.push({0, 1, 0});

    long long total = 0;
    std::vector<std::tuple<int, int, long long>> mst_edges;
    mst_edges.reserve(n - 1);

    while (!pq.empty()) {
        auto it = pq.top();
        pq.pop();
        int v = it.v;
        if (used[v]) continue;
        used[v] = true;
        parent[v] = it.p;
        edge_w[v] = it.w;
        if (it.p != 0) {
            mst_edges.emplace_back(it.p, v, it.w);
            total += it.w;
        }

        for (const auto& e : g[v]) {
            if (!used[e.to]) {
                pq.push({e.w, e.to, v});
            }
        }
    }
}
```
